## Modules and Tools

This project consists of the following core solutions:

### Visual Database Designer
An intuitive browser-based tool where you can drag and drop tables to create relationships visually. Once your design is complete, the tool exports the exact script needed to implement it in your system, supporting both relational databases and collection-based models.

### Fair Task Allocator
A web application designed for team leaders and project managers. By inputting the to-do list and the team members, the platform analyzes and suggests the best way to distribute the workload, ensuring an even distribution and preventing burnout.

### Visual Branch Synchronizer
A web tool that connects directly to GitHub to monitor your repository. It provides visual alerts in your development environment (like Visual Studio) if there are new changes in the main branch that you need to pull before continuing to code, saving you from headaches and merge conflicts.

### Rapid Documentation Generator
A smart utility that reads your source code and automatically generates a clean, well-structured README.md file. It clearly explains the purpose of each main component of your solution.

### AI-Powered Code Review
A code review system driven by artificial intelligence agents. It features an in-app "MCP" that connects with your GitHub to:
- Detect areas for improvement in your code.
- Suggest optimization and readability enhancements.
- Apply recommended modifications directly from the app, facilitating the adoption of coding best practices.
